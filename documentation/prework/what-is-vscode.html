What is VS Code?

VS Code: Your coding workspace
Visual Studio Code is a free text editor designed for writing code and HTML files. Think of it as Microsoft Word, but for code.

Notepad

Plain text, no hints, easy to make typos

VS Code

Color coding, auto-complete, error detection

Open your project folder
Always open the entire project folder, not individual files.
  • File → Open Folder...
  • Select "wwf-discovery-day"
  • Click "Open"
Why a folder? Opening the whole folder lets you see all project files in the sidebar and enables Git integration.
The interface: sidebar + editor
Left side shows your files. Right side is where you edit. Click any file to open it.

Edit files like normal
Just click and type! VS Code color-codes your HTML automatically. Save with Cmd+S (Mac) or Ctrl+S (Windows).
