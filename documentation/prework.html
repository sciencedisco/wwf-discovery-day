Prework & Prerequisites

You'll set up and install some applications, test that you have all necessary components by running a notebook, and then verify you did it all by submitting a short form. This whole process should take 30–45 minutes.

Setup & Requirements

Estimated time: 15–30 minutes

Set up each component below. Work through them in order, and check off each one as you complete it.

Already have some of these? If you already have GitHub, VS Code, Git, Python, or any other tool installed, feel free to skip those steps.

1

Create a GitHub Account

What is it? GitHub is where you'll store and share your code.

  • Go to github.com and sign up with a personal email you'll have forever (not your school email)
  • You'll create a GitHub username (or "handle") that you'll need to share later
Once you have a GitHub account and username, move to the next step
2

Get Free GitHub Copilot (Student Access)

What is it? GitHub Copilot is an AI assistant that helps you write and understand code.

  • Go to GitHub Student Developer Pack
  • Click "Sign up for student developer pack" and during the application, you'll be instructed to link your school email to verify you're a student
  • You'll need to wait a few days for approval, but you can continue with other steps in the meantime. You'll just have limited access to GitHub Copilot until approved
Once your application is submitted, move to the next step
3

Download and Install VS Code

What is it? VS Code is the code editor where you'll write code and run notebooks.

  • Go to code.visualstudio.com and download for your operating system
  • Run the installer and follow the default steps
  • Open VS Code to make sure it works
Once VS Code opens, move to the next step
4

Install GitHub Copilot in VS Code

What is it? This adds the AI assistant to your code editor (requires GitHub Copilot approval from step 2).

  • In VS Code, click the Extensions icon on the left sidebar (looks like four squares)
  • Search for "GitHub Copilot" and click "Install"
  • Sign in with your GitHub account when prompted
  • Once installed, open Copilot Chat by clicking the chat bubble icon in the left sidebar, or press Ctrl+Alt+I (Windows) / Cmd+Alt+I (Mac)
Once Copilot is installed and active, move to the next step
5

Install Git

What is it? Git is a tool that tracks changes to your code and lets you upload it to GitHub.

  • Go to git-scm.com and download for your operating system
  • Run the installer and use the default settings
  • In VS Code, open a terminal via the top menu: Terminal → New Terminal, then type:
    git --version

💡 Stuck? Paste any error message into Copilot Chat and ask for help.

The terminal should show something like git version 2.x.x — if you see a version number, you're good to go.
6

Install Python

What is it? Python is a programming language. You'll need it to run the notebooks and code during Discovery Day.

  • Go to python.org and download the latest version for your operating system
  • Run the installer and use the default settings
  • In VS Code, open a terminal via the top menu: Terminal → New Terminal, then type:
    python --version

💡 Stuck? Paste any error message into Copilot Chat and ask for help.

The terminal should show something like Python 3.x.x — if you see a version number, you've completed all setup!

Complete Your Prework

Estimated time: 10–15 minutes

Follow these three steps to complete your prework. Go at your own pace and don't worry if you get stuck—that's totally normal.

Step 0: Download the Test Notebook

First, download the notebook file we've prepared for you. This is the file you'll run in VS Code.

↓ Download Notebook

Save this file to your computer. Remember where you save it!

1

Run the notebook in VS Code

In this step, you'll open the notebook you just downloaded and run it to see the charts and data.

Here's how:

  • Open VS Code and drag the notebook file into it (or use File → Open File)
  • Wait a few seconds for VS Code to recognize it's a Jupyter notebook
  • Click the "Run All" button at the top (or use Ctrl+Enter on each cell to run them one by one)
  • Watch as cells execute. You should see colorful charts and tables appear
  • If something breaks, that's okay! Click on the cell with the error, then open Copilot Chat and ask: "This cell is giving an error — can you help me fix it?" Paste in the error message for best results
Once you see at least one chart or table, move to Step 2
2

Use GitHub Copilot to improve something

Now you'll use AI to make a small improvement. This is the heart of the prework—learning to work with Copilot.

Here's how:

  • Open Copilot Chat in VS Code (Ctrl+Shift+I on Windows, Cmd+Shift+I on Mac)
  • Ask Copilot a real question about what you see, for example:
  • "How can I make this chart title more descriptive?"
  • "What does this code do in plain English?"
  • "Can you suggest a better color scheme for this chart?"
  • Read what Copilot suggests. If it makes sense to you, apply the change to the notebook
  • Run the notebook again (Run All) to make sure your change works. Does it look better?
Once you've made one change and run the notebook again, you're ready for Step 3
3

Capture your work

Last step! Take a screenshot showing both your notebook results and the Copilot chat panel.

Here's how:

  • Arrange your screen so you can see both the notebook with your results AND the Copilot chat panel in one view
  • Take a screenshot (Mac: Cmd+Shift+4, Windows: Win+Shift+S, or just use Print Screen)
  • Save the screenshot somewhere easy to find on your computer
Great! You've completed all three steps. Time to submit.
