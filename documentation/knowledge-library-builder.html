Knowledge Library Builder

← Back to Overview

Design your interactive knowledge library structure. Define search terms, filters, facets, display fields, and how each knowledge item is structured. This helps you plan the complete search and discovery experience before you build!

Static Filters

Filters that are ALWAYS visible regardless of search term. Examples: "Data Type", "Time Period", "Published Year"

Dynamic Facets

Context-dependent filters shown only when relevant to the selected search key term. Example: "Conservation Status" facet only appears when searching "Jaguar" or "River Dolphins".

Key Terms

Define the predefined search queries users can select from the search bar. Note: Free-format text search is outside the scope of this exercise. Users will only be able to select from these predetermined key terms.

Examples: "Jaguar Population", "Deforestation Trends", "River Dolphin Conservation", "Indigenous Communities"
