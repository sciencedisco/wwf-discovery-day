Research Questions

← Back to Overview

As a student, you'll work on one of the following research challenges using the available datasets. Choose the question that excites you most, then design and build a data-driven solution to answer it.

Option 1

Forest Biome Dashboard

Create a forest biome dashboard based on external data that presents 3 key figures for each of the following three questions:
  1. What is the current state of biodiversity and the urgency for action?
  2. What external pressures and drivers affect conservation work?
  3. Which trends and opportunities for conservation are emerging and what future outlook can be expected?

A 3×3 dashboard with 3 key figures for each of the 3 questions

Option 5

Amazon Knowledge Library

Create an interactive knowledge library for the Amazon region based on the provided factsheet.

The knowledge library should:

  • Provide an overview and relevant figures on a specific topic requested by the user
  • Indicate if information is missing or unavailable
  • Allow users to add additional information and sources to the library
  • Be interactive and searchable

Think of it as a living, collaborative knowledge base for the Amazon region
