What You'll Need

To participate fully in Discovery Day, you'll need access to a few key tools. Don't worry—everything is free for students!

  • GitHub Account - For version control and hosting your dashboard
  • VS Code - Code editor with GitHub Copilot integration
  • GitHub Copilot - AI pair programmer (free for students)
  • Python - For data analysis in Jupyter notebooks
